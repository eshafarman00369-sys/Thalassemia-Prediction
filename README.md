# Alpha Thalassemia Disease Prediction Website

A frontend website for predicting Alpha Thalassemia disease risk using medical parameters.

## Features
- User-friendly interface for medical data input
- Real-time disease risk prediction
- Responsive design for all devices

## Technologies Used
- HTML5
- CSS3  
- JavaScript

## How to Use
1. Open `index.html` in your browser
2. Enter medical parameters:
   - Hemoglobin level
   - RBC count
   - MCV level
   - Age
3. Click "Predict Disease" to see results

## Note
This is a frontend demonstration. For accurate medical diagnosis, consult healthcare professionals.
